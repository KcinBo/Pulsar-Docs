## GitBook image use

Pages inside `docs/` reference assets with a relative path beginning `../.gitbook/assets/`. The root `README.md` uses `.gitbook/assets/`. Preserve this directory layout when importing or syncing the repository.

These files preserve the supplied branding. Public usage rules, trademark guidance, minimum sizes, spacing, and approved color values can be added here once finalized.
