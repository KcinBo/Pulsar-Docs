# Package contents

This is the corrected multi-page Pulsar Protocol GitBook package.

- `.gitbook.yaml` points GitBook to the repository root, landing page, and navigation file.
- `README.md` is the documentation landing page.
- `SUMMARY.md` defines the complete page tree.
- `docs/` contains one Markdown file per documentation page.
- `assets/` is a normal, visible folder containing the supplied full logo, transparent logo, wordmark, icon, and brand board.
- `IMPORT-INSTRUCTIONS.md` explains repository import and image handling.
- Mathematical notation uses GitBook-native KaTeX delimiters and commands.

There is intentionally no all-in-one documentation file. This prevents a single combined page from replacing the intended page tree during import.
